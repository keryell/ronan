En général les fichiers trans_expose.pdf sont ceux en couleur que j'utilise lors de la présentation et les fichiers trans_eleves.ps sont ceux que je distribue aux élèves et possèdent en fait 2 transparents noir et blanc par page papier.

Le texte des travaux pratiques éventuels se trouve dans TP.ps et les corrections dans corrige_TP.ps.

Les transparents sont faits avec le style seminar et hyperref pour LaTeX et transformés en PDF via gs (version 6.01 au moins...) pour affichage par projecteur vidéo avec acroread en mode pleine page avec transition entre les pages par « Glitter-Right-Down ». Typiquement mon Makefile ressemble à : 

all : trans.pdf

bib : article.bbl

#INSTALL_DIR = $(HOME)/public_html/publications/ENSTBr_INFO_2000-001

install :
	mkdir -p $(INSTALL_DIR)
	mv article.pdf $(INSTALL_DIR)

BIBLIO = $(HOME)/TeX/bib/biblio.bib

LATEX=frlatex

%.dvi : %.tex
	$(LATEX) $<

%.bbl : %.tex $(BIBLIO)
	bibtex $*

%.ps : %.dvi
	dvips $< -o


%.pdf : %.ps
	ps2pdf $<

%.bib : %.bob
	cherche_tox $*

trans.ps : trans.dvi Makefile
	# Faire le bon choix ici :
	# PDF :
	dvips -O-1mm,81mm -T205mm,290mm trans.dvi -o
	# Copie papier double :
	#dvips -O0mm,5mm trans -o
	# Papier :
	#dvips -O2.5mm,3mm trans.dvi -o

trans.pdf : trans.ps Makefile
	# Faire le bon choix ici :
	## PDF :
	gs -q -dNOPAUSE -dBATCH -sDEVICE=pdfwrite -g8350x5900 -sOutputFile=$@ trans.ps
	## Copie papier double :
	#ps2pdf $*.ps $*.pdf

%.html : %.tex
	latex $*
	latex $*
	latex $*
	latex $*
	tex4ht $*
	t4ht $*
	# Corrige les "»" :
	sed 's/»/\»/g' $*.html > $*.html.$$ && mv $*.html.$$ $*.html

clean:
	rm  *.aux *.dvi *.log *.los *.pdf *.ps
