\subsection{Contexte}
\label{sec:contexte}

Ce cours constitue le module \Aiahp (Informatique À Haute Performance) de
la \href{https://info.enstb.org/enseignement/mri}{branche brestoise} du
\AdeainforennesI avec le module \Aiahp (ex-\Aapco, architecture
parallélisme et compilation) et de l'\Aisia, Institut Supérieur
d'Informatique et d'Automatique de l'École des Mines de Paris.


\subsection{Mots-clés}
\label{sec:mots-cles}

Architecture des ordinateurs, parallélisme, optimisation de programmes,
compilation, langages parallèles, programmation parallèle, systèmes
embarqués.


\subsection{Objectifs}
\label{sec:objectifs}

% Bug TeX4ht
~
\begin{itemize}
\item Présentation des techniques utilisées dans les ordinateurs pour
  atteindre les performances les plus élevées d'un point de vue
  \begin{itemize}
  \item architecture (parallélisme, systèmes embarqués) ;
  \item langage ;
  \item compilation ;
  \item applications ;
  \end{itemize}    
\item utilisation industrielle des ordinateurs parallèles.
\end{itemize}    
    
    
\subsection{Prérequis}
\label{sec:prerequis}

Notions d'architecture des ordinateurs, programmation en langage C,
\Aunix.


\subsection{Logiciels nécessaires}
\label{sec:logic-necess}

Environnement \Aunix, \Ampi, \Aopenmp, compilateur \Afortran ou C avec
parallélisation \Aopenmp.


\section{Contenu}
\label{sec:contenu}

Le module est constitué de 18 heures de cours et 12 heures de travaux
pratiques. L'emploi du temps se trouve à
\LienPDF{https://info.enstb.org/enseignement/mri/planning}.

La fiche de description du cours à l'\Aisia se trouve à :\\
\LienPDF{http://www-isia.cma.fr/liste_cours/I\%2034\%20Informatique\%20\%E0\%20Hautes\%20Performances.pdf}
  et la liste de tous les cours dans
  \LienPDF{http://www-isia.cma.fr/page.php?rub=enseignements&srub=cours}.


\subsection{Liste du contenu}
\label{sec:liste-du-contenu}

% Bug TeX4ht
%~
\begin{itemize}
\item \LienPDFtexte{\InstallDir/Introduction}{Introduction} ;
\item \LienPDFtexte{\InstallDir/Archi_Avancee}{architecture avancée et
    parallélisme} (2 jours) :
  \begin{itemize}
  \item intra-processeur (pipeline, superscalaire, \Avliw, simultaneous
    multi-threading, fonctions booléennes) ;
  \item parallélisme inter-processeur (\Asimd, \Amimd, \Aasip, flot de
    donnée, réseaux d'interconnexion, entrées-sorties,...) ;
  \item vous avez dit performances ? Systèmes enfouis ou embarqués (\Asoc,
    \Anoc,...) ;
\end{itemize}
\item \LienPDFtexte{\InstallDir/Optimisation_Code}{optimisation de
    programmes séquentiels} ;
\item programmation
  \begin{itemize}
  \item à parallélisme de données (\Afortran \Ahpf) ;
  \item à parallélisme de contrôle (\Aopenmp, \Ampi) ;
  \item \Afortran, C et langage machine ;
\end{itemize}
\item étude de la parallélisation d'un code de simulation sur divers
  systèmes :
  \begin{itemize}
  \item parallélisation sur ordinateurs mémoire partagée (type Sun
    multiprocesseur, Silicon Graphics) avec \Aopenmp ;
  \item parallélisation sur système \Ampp en mode passage de message avec
    \Ampi et cluster de \Apc \Alinux
  \end{itemize}
\item visite à \Amecalog : application du parallélisme aux crashes virtuels
  de voitures : présentation
  \begin{itemize}
  \item des techniques de programmation des langages,
  \item de l'application,
  \item des machines utilisées.
  \end{itemize}
\end{itemize}
